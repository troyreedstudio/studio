### Balenciaga → AGAPĒ replacement method (for future use)
1. Extract bottom strip: `sharp().extract({ left:0, top:750, width:1206, height:148 })`
2. Use image tool to find exact text coords in strip
3. Add strip y-offset (750) to get absolute coords
4. Composite: dark rect cover first, then AGAPĒ SVG text centered over it
5. Cover fill colour: `#060810` (matches swimsuit dark navy)
6. Always rename output file (story_v5, v6 etc) to bust Cloudflare cache

## ⚠️ CRITICAL — Product Logo Placement Rule (Locked by Sascha, 2026-03-19)

**NEVER use DALL-E 3 or any AI image generator to render the AGAPĒ logo/text on products.**
AI models cannot reliably spell or render our exact wordmark — they always get it wrong.

**ALWAYS use this two-step pipeline for product mockups:**
1. Generate a blank, high-quality product image using **gpt-image-1** (NOT dall-e-3)
   - Prompt must say: "NO text, NO logos, NO embroidery, NO lettering — completely blank"
   - Use gpt-image-1 for photorealistic quality
2. Composite our exact AGAPĒ SVG/path onto the product using **sharp**
   - Use text-to-svg → path → sharp composite
   - This guarantees pixel-perfect, correct typography every time

**This applies to:** towels, caps, hoodies, vests, bags, any product placement.
